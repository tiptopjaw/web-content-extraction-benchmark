# Modern Content Extractor Benchmark

A benchmark for evaluating content extraction algorithms using 1,193 modern web pages (2025) with AI-generated ground truth.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/tiptopjaw/text-extraction-benchmark.git
cd text-extraction-benchmark

# Extract HTML files
7z x data/html_files.7z -odata/

# Install dependencies
pip install -r requirements.txt

# Run benchmark
python scripts/03_run_benchmark.py
```

## Dataset

- **1,193 annotated web pages** across diverse content types
- **Ground truth** generated by DeepSeek API with quality filtering
- **HTML sources** included (71MB compressed, 704MB extracted)

### Content Types

| Type | Coverage |
|------|----------|
| Informational articles | ~40% |
| Blog posts/guides | ~30% |
| Product pages | ~15% |
| Service pages | ~10% |
| Other | ~5% |

## Ground Truth Format

```json
{
  "url": "https://example.com/article",
  "file_id": "0001",
  "ground_truth": {
    "title": "Article Title",
    "author": "John Doe",
    "publish_date": "2025-01-15",
    "main_content": "Full article text...",
    "with": ["Sentences that MUST be extracted"],
    "without": ["Boilerplate that should NOT be extracted"]
  }
}
```

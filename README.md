# Modern Content Extractor Benchmark

A benchmark for evaluating content extraction algorithms using 1,507 modern web pages (2025-2026) with AI-generated ground truth.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/Murrough-Foley/web-content-extraction-benchmark.git
cd web-content-extraction-benchmark

# Extract HTML files
7z x data/html_files.7z -odata/

# Install dependencies
pip install -r requirements.txt

# Run benchmark
python scripts/03_run_benchmark.py --extractor rs-trafilatura
```

## Dataset

- **1,507 annotated web pages** across diverse content types
- **Ground truth** generated by DeepSeek API with quality filtering
- **HTML sources** included (compressed archive)

### Content Types

| Type | Count | Description |
|------|-------|-------------|
| Article | 842 | News, blog posts, guides |
| Unknown | 398 | Unclassified pages |
| Service | 151 | Service descriptions |
| Collection | 91 | Curated lists, roundups |
| Documentation | 30 | Technical docs |
| Forum | 25 | Discussion threads |
| Review | 17 | Product/service reviews |
| Product | 13 | Product descriptions |

## Ground Truth Format

```json
{
  "url": "https://example.com/article",
  "file_id": "0001",
  "ground_truth": {
    "title": "Article Title",
    "author": "John Doe",
    "publish_date": "2025-01-15",
    "main_content": "Full article text...",
    "with": ["Sentences that MUST be extracted"],
    "without": ["Boilerplate that should NOT be extracted"]
  }
}
```

## Benchmark Results

| Extractor | Precision | Recall | F1 Score |
|-----------|-----------|--------|----------|
| rs-trafilatura | 0.622 | 0.871 | 0.688 |
| boilerpy3-article | 0.513 | 0.722 | 0.560 |
| boilerpy3-default | 0.482 | 0.740 | 0.550 |
| beautifulsoup | 0.178 | 0.959 | 0.285 |

*Results on 1,507 files as of January 2026.*

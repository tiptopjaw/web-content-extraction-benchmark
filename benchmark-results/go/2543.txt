Overview This project demonstrates how to create a simple PL design in Vivado to control the user-defined LEDs (DS7 and DS8) on the KR260 carrier board. The design uses an AXI GPIO IP core connected to the Zynq UltraScale+ Processing System (PS) via an AXI SmartConnect, enabling software control of the LEDs from Linux running on the APU. This is a follow-up project to the Custom Yocto Linux & Network Boot for Xilinx KR260 post, which provides the foundation Linux environment with PYNQ, XRT, and network boot capabilities. development workflows require: Writing HDL code or using IP integrator Synthesizing and implementing the design Generating bitstreams Flashing SD cards or manually loading bitstreams Writing C/C++
